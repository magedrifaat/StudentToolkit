This is CS50's final project!

This website aims at helping students by bringing together some of the tools found on the internet that students frequently use from vocabulary searching , spell and grammar checking, to solving equations and graphing functions. In addition to a page for some external sources for education that I, as a student, find them really helpful.


    APIs used in this project:
  • Bing spell check API from Microsoft Azure.
  • Grammar API from Language Tool.
  • Dictionary API from Oxford Dictionaries.
  • Full results API from Wolfram|Alpha.

    Technologies used in this project:
  • HTML, CSS for front-end designing.
  • JavaScript with JQuery for front-end development.
  • Python with Flask microframework for back-end development.
